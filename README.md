# 🌀 OBITO-MD (Uchiha Edition)

![Obito Uchiha](https://wallpapercave.com/wp/wp8955214.jpg)

**OBITO-MD** est un bot WhatsApp ultra-performant conçu pour Replit, utilisant le système de jumelage direct (**Pairing Code**). Fini les QR codes complexes, connectez-vous instantanément avec votre numéro.

## ⚡ Caractéristiques
- **Connexion Kamui** : Jumelage par code rapide et sécurisé.
- **Interface Uchiha** : Un design web immersif inspiré du Mangekyou Sharingan.
- **Auto-Clean** : Nettoyage automatique des sessions pour éviter les erreurs de connexion.
- **Fast-Flow** : Réponse instantanée aux commandes.

## 🛠 Installation sur Replit

1. **Fichiers** : Assurez-vous d'avoir les fichiers à la racine :
   - `index.js` (Le cerveau)
   - `index.html` (L'interface visuelle)
   - `package.json` (Les dépendances)
   - `README.md` (Ce fichier)

2. **Configuration** :
   - Cliquez sur le bouton **Run**.
   - Ouvrez la fenêtre Webview.
   - Entrez votre numéro WhatsApp (format international : 243...).

3. **Lien WhatsApp** :
   - Copiez le code affiché.
   - Sur votre téléphone : `WhatsApp > Appareils connectés > Connecter avec le numéro`.
   - Entrez le code généré.

## 🌀 Commandes de base
- `.menu` : Affiche la liste des fonctionnalités.
- `.ping` : Vérifie la réactivité du bot.
